# 🏏 IPL 2026 Player Performance Analytics Dashboard

## 📌 Overview

The IPL 2026 Player Performance Analytics Dashboard is an interactive web-based dashboard designed to analyze player and team performance using a self-created Excel dataset. It provides insights into batting, bowling, team standings, player salaries, and tournament performance through interactive charts and filters.

> **Note:** This project uses a fictional dataset created for learning and portfolio purposes.

---

## 🚀 Features

- Executive KPI Dashboard
- Team Performance Analysis
- Player Performance Analysis
- Top Run Scorers
- Top Wicket Takers
- Salary Distribution Analysis
- Team Points Table
- Interactive Filters
- Search Functionality
- Responsive Dashboard Design

---

## 🛠️ Tech Stack

- Microsoft Excel (Dataset)
- HTML5
- CSS3
- JavaScript
- Chart.js
- Claude AI (AI-assisted dashboard development)

---

## 📊 Dataset

The dataset was manually created in Microsoft Excel and includes:

- Team Name
- Player Name
- Player Role
- Runs
- Wickets
- Salary
- Fours
- Sixes
- Dot Balls
- Catches
- Strike Rotation
- Tournament Points

---

## 🎯 Project Objectives

- Analyze player performance across IPL teams.
- Compare team performance using interactive visualizations.
- Identify top performers based on batting and bowling statistics.
- Demonstrate dashboard design and data visualization skills.

---



## ▶️ How to Run

1. Download or clone the repository.
